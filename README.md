# DIO_NotebookLM
Repositório destinado a apresentar o conhecimento adquirido com o desafio de projeto de uso do NotebookLM do Google.

---

### Link do Notebook: https://notebooklm.google.com/notebook/7965aeae-3887-4248-a8ce-6c54d3b25645

---

A base para esse Notebook foi a franquia de jogos Nier, da Square Enix. Foram utilizados como fontes para o Notebook vídeos explicativos da história dos jogos, a wiki e outros artigos relacionados.

Escolhi Nier como base para o Notebook pois é algo que eu tenho um conhecimento considerável sobre.
